HTMLMinifier: Online HTML/JS/CSS Minifier 


        


        

          

            HTMLMinifier is an excellent tool to help you minify your HTML/JavaScript/CSS! Uses
            html-minifier-terser
            which comes with
            terser and clean-css.
          

          
This is a simple wrapper for the browser around the minifier; the hard work is done by the terser team and clean-css team.

          

            
              This wrapper for the browser was created by
              coo11. Inspired by
              UglifyJS 3: Online JavaScript minifier.
              

              html-minifier-terser {HMT_VERSION}
              

              terser {TERSER_VERSION}
              

              clean-css {CCSS_VERSION}
